### Run Locally

To run **Richard Ryan** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/persiandarik/Portfolio-Photographer.git
```

Windows:

```bash
git clone https://github.com/persiandarik/Portfolio-Photographer.git
```
